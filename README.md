# Tetris-project
# Browser-Based Tetris Game

A web-based implementation of the classic Tetris game where users can play directly in their browser and compete for high scores through a leaderboard system.

## Screenshot
![Tetris Game Screenshot]

## Features

- Playable Tetris game in the browser
- User registration and login
- Leaderboard displaying top scores
- Interactive game interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Object-Oriented Programming

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in a web browser.

## Project Overview

This project was developed as part of a Web Applications and Databases module at university. The aim was to build an interactive browser-based game while implementing features such as user authentication and a ranking system.
